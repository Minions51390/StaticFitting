# static_fitting

## 建议node版本12以上

## 无法翻墙建议修改npm和yarn源（推荐使用yarn）
```
yarn config set registry https://registry.npm.taobao.org
npm config set registry https://registry.npm.taobao.org
```


## 安装项目依赖
```
yarn install
npm install
```

### 运行项目
```
yarn serve
npm run serve
```

### 打包项目
```
yarn build
npm run serve
```

### proxy修改
```
在vue.config.js文件devServer对象下proxy
```
